### General information 

This project not yet be finished, and still not have real blog only placeholder it will be fix soon

There might be things broken in other web browsers, tested only on Chrome.


This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

First, run the development server:
 
```bash
npm run dev
# or
yarn dev
## and ##
rename .env.example to .env and insert you api_key
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `pages/index.js`. The page auto-updates as you edit the file.
